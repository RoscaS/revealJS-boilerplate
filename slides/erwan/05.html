Partager c'est vertueux



Une classe partagée consiste en un bloc de code partagé qui contient les données et un compteur de partages.

Quand l’objet est crée le compteur est initialisé à 1. A chaque fois qu’on le copie on l’incrémente, et à chaque fois qu’une référence est détruite on le décrémente.

La données partagée est détruite si le compteur atteint 0.
